# uoko-fe-data-platform

## 项目介绍

优客逸家 - 数据平台移动版(Typescript)

## 简介

项目为移动端Web，核心采用Vue 2.x版本，开发语言为 `TS`。

## 项目安装

克隆地址：

http://win-tfs-cd-01.uoko.local:8080/tfs/DefaultCollection/star/_git/uoko-fe-data-platform

安装依赖：

```bash
npm install
```

### 在开发模式下启动并带有热更新

```bash
npm run serve
```

### 生产环境打包

```bash
npm run build
```

### 测试

```bash
npm run lint
```

### 运行单元测试

```bash
npm run test:unit
```

### 运行E2E测试

```bash
npm run test:e2e
```
